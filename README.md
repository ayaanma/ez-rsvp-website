# e-z.rsvp

**RSVP now. Find out where later.**

e-z.rsvp is a modern mystery-event RSVP web app that makes going out easier, more spontaneous, and less stressful. Users choose their timing, budget, location radius, and broad preferences, then discover the exact event closer to the start time.

The core idea is simple: planning is hard. We make it e-z.

---

## Overview

e-z.rsvp lets users browse and RSVP to surprise local events. Instead of showing every detail upfront, the app keeps the event partially hidden until reveal time. Users can still set hard preferences like location radius, price, category, and group plans so the experience feels fun but safe.

e-z.rsvp is for people who want to go out but do not want to spend hours planning. Users choose the boundaries. The app handles the plan.

The app includes:

- A sleek light-theme homepage
- Google/Apple-only authentication pages
- My Events dashboard
- Find Events search/filter page
- Mystery mode toggle
- Event detail overlays
- Group planning page
- Account settings page
- Organizer submission page
